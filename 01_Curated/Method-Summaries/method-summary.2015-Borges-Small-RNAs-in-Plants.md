# Methods extracted from Paper

Source paper: `2015-Borges-Small-RNAs-in-Plants`

> 注意：本文是综述，以下为文中使用或凝练出的“综述/通路/机制框架方法”，不是作者在本文中新做的实验方法。

## Method list

### Method 1: Pathway-centered literature synthesis

- Method name: 小 RNA 通路中心化文献综合
- Purpose: 按 miRNA、secondary siRNA、heterochromatic siRNA 等通路组织植物小 RNA 知识。
- Input material/data: 已发表的小 RNA 生物发生、功能和表观遗传研究；本文未报告新数据。
- Output: 植物小 RNA 通路框架和功能场景总结。
- Key parameters: DCL, RDR, AGO, small RNA length, precursor type, PTGS/TGS, RdDM。
- Strengths: 能把分子机制与发育/表观遗传功能连接。
- Limitations: review-level evidence；具体实验强度依赖被引论文。
- Used in which result: Finding 1, Finding 2, Finding 6。

### Method 2: AGO sorting framework

- Method name: AGO loading/sorting 机制框架
- Purpose: 解释小 RNA 如何装载到不同 AGO 并产生功能分化。
- Input material/data: 小 RNA 5' terminal nucleotide、miRNA duplex mismatch/bulge 结构、AGO 偏好文献。
- Output: AGO1/AGO2/AGO4/AGO6/AGO7/AGO9/AGO10 等 sorting 规则摘要。
- Key parameters: 5' nucleotide identity, MID domain specificity, duplex mismatch positions, guide/passenger strand structure。
- Strengths: 将小 RNA 序列/结构特征与功能输出连接。
- Limitations: 具体规则多来自 Arabidopsis 案例；其他植物外推需谨慎。
- Used in which result: Finding 3。

### Method 3: Small-RNA modification model

- Method name: 小 RNA 3' 端修饰与稳定性模型
- Purpose: 总结 2'-O-methylation、uridylation 与 degradation 如何调控小 RNA 稳定性和活性。
- Input material/data: HEN1、HESO1、URT1、SDN 相关文献。
- Output: HEN1 保护、HESO1/URT1 尿苷化、SDN 降解的机制框架。
- Key parameters: 3' terminal methylation, uridylation, small RNA length variants, AGO1 association。
- Strengths: 有助于解释 small RNA abundance 和 non-canonical function。
- Limitations: MinerU 对化学符号解析噪声较大；needs manual PDF check。
- Used in which result: Finding 4。

### Method 4: Secondary siRNA trigger model

- Method name: secondary siRNA one-hit/two-hit trigger model
- Purpose: 解释 miRNA cleavage 如何触发 RDR6-dependent secondary siRNA production。
- Input material/data: miRNA target cleavage、transcript templates、RDR6/SGS3/DCL4/DCL2 机制文献。
- Output: tasiRNA、phasiRNA、easiRNA 生成路径。
- Key parameters: 21 nt two-hit miRNA, 22 nt one-hit miRNA, RDR6, SGS3, DCL4, DCL2, phasing。
- Strengths: 适合在 claim-evidence 表中区分触发机制和功能结果。
- Limitations: phasiRNA 功能在许多物种中仍未明确；不能把 phasing 直接等同于已知功能。
- Used in which result: Finding 5。

### Method 5: PTGS-to-TGS transition / RdDM pathway model

- Method name: PTGS 到 TGS 转换与 RdDM 通路模型
- Purpose: 描述活化转座子、epialleles 或 transgenes 如何从 21/22 nt siRNA 相关 PTGS 转向 24 nt siRNA/RdDM 介导 TGS。
- Input material/data: Pol II/Pol IV/Pol V、RDR6/RDR2、DCL2/DCL4/DCL3、AGO6/AGO4、DRM2 相关文献。
- Output: 21/22 nt siRNA 到 24 nt siRNA、DNA methylation 和 chromatin silencing 的概念模型。
- Key parameters: Pol II transcription, RDR6, DCL2/DCL4, DCL3, Pol IV, Pol V, RDR2, AGO4/AGO6/AGO9, DRM2。
- Strengths: 连接 post-transcriptional 和 transcriptional silencing。
- Limitations: 作者明确指出部分 interplay is not fully understood；需避免过度因果化。
- Used in which result: Finding 6。

### Method 6: Developmental-context synthesis

- Method name: 发育与繁殖情境中的小 RNA 功能综合
- Purpose: 汇总 small RNA 在 meiosis、gametogenesis、genomic imprinting、paramutation、hybrid incompatibility 等场景中的作用。
- Input material/data: 被引植物发育、繁殖、表观遗传文献。
- Output: 以组织/发育阶段为坐标的小 RNA 功能图景。
- Key parameters: reproductive tissue, gametophyte, phasiRNA, AGO5/MEL1/AGO9/AGO104, mobile small RNA, RdDM。
- Strengths: 提供从分子通路到发育功能的桥梁。
- Limitations: 多处功能仍 unknown/enigmatic；review-level evidence。
- Used in which result: Finding 6。
