## Method list

### Review synthesis of RdDM literature

- Method name: Narrative review and pathway synthesis
- Purpose: 整合植物 RdDM canonical pathway、变体、靶向机制和生物学功能。
- Input material/data: published genetic, biochemical, sequencing, methylome and chromatin studies.
- Output: conceptual pathway model; Table 1 component inventory; discussion of unresolved questions.
- Key parameters: not applicable.
- Strengths: 提供通路级知识框架。
- Limitations: review-level evidence; underlying primary papers must be read for formal evidence grading.
- Used in which result: all findings.

### RdDM-defective mutant analysis

- Method name: Mutant-based pathway component identification
- Purpose: 识别 Pol IV、Pol V、RDR2、DCL3、AGO4、DRM2 等 RdDM 组分。
- Input material/data: Arabidopsis mutants defective in RdDM-related genes.
- Output: loss of 24-nt siRNAs, altered DNA methylation, defective transcriptional gene silencing.
- Key parameters: varies by cited study; not reported in full detail in this review text.
- Strengths: genetic evidence can assign pathway components.
- Limitations: reviewed second-hand; exact alleles and assays require primary paper check.
- Used in which result: Table 1; canonical RdDM pathway.

### Small RNA sequencing

- Method name: sRNA-seq
- Purpose: 测量 24-nt siRNA abundance and distribution in wild type and mutants.
- Input material/data: small RNA libraries from plant tissues/genotypes.
- Output: 24-nt siRNA profiles and Pol IV-dependent siRNA loss patterns.
- Key parameters: not reported in provided text.
- Strengths: genome-wide resolution of siRNA biogenesis.
- Limitations: review-level description only.
- Used in which result: Pol IV-dependent siRNA biogenesis.

### Genome-wide DNA methylation profiling

- Method name: methylome profiling at single-nucleotide resolution
- Purpose: 比较 wild-type and mutant plants 中 CG/CHG/CHH methylation patterns。
- Input material/data: genomic DNA from plants.
- Output: methylation maps and RdDM target identification.
- Key parameters: next-generation sequencing highlighted by the review.
- Strengths: 可全基因组定位 RdDM 作用区域。
- Limitations: technology and thresholds vary across cited studies.
- Used in which result: RdDM target preferences; chromatin interplay.

### ChIP-seq and chromatin feature mapping

- Method name: Chromatin immunoprecipitation followed by sequencing
- Purpose: 定位 Pol V occupancy and chromatin marks such as H3K9me。
- Input material/data: chromatin from Arabidopsis tissues/genotypes.
- Output: Pol V-bound sites, chromatin-mark associations, target preferences.
- Key parameters: not reported in provided text.
- Strengths: links RdDM machinery to chromatin context.
- Limitations: Pol V occupancy alone is not sufficient for RdDM at all sites.
- Used in which result: Pol V-mediated de novo methylation; SHH1 recruitment discussion.

### Protein interaction and biochemical assays

- Method name: Protein interaction/biochemical characterization
- Purpose: 解析 AGO4, NRPE1, KTF1, RDM1, SUVH2/9 and other proteins in RdDM complex assembly.
- Input material/data: proteins/domains from Arabidopsis RdDM components.
- Output: interaction models, chromatin-binding models, scaffold recruitment hypotheses.
- Key parameters: not reported in provided text.
- Strengths: supports mechanistic interpretation beyond mutant phenotypes.
- Limitations: review-level summary; biochemical conditions require primary paper check.
- Used in which result: Pol V scaffold and silencing effector complex model.
