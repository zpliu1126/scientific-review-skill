# Example: Review Paragraph

## Draft Paragraph

Transcriptome studies provide a useful entry point for identifying drought-responsive candidate genes in crops. In the provided abstract for Paper A, drought treatment was associated with differentially expressed genes enriched in ABA and osmotic stress terms, and selected expression trends were checked by qRT-PCR/qPCR. User notes for Paper B report a WGCNA module correlated with a drought tolerance index and list candidate hub transcription factors. Together, these sources support the model synthesis that drought response reviews can distinguish expression-based candidate discovery from functional validation. However, neither source by itself demonstrates that the candidate genes causally improve drought tolerance; that claim would require perturbation and phenotype evidence. [source needed]

## Source-Reported Content Used

- Paper A: drought-responsive DEGs, enriched stress-related terms, and selected qRT-PCR/qPCR validation.
- Paper B: WGCNA module correlated with drought tolerance index and candidate hub transcription factors.

## Reasonable Inference Used

- These sources support candidate gene prioritization but not causal gene-function claims.

## Model Synthesis Used

- A review can organize drought transcriptome evidence by separating expression association, co-expression, qRT-PCR/qPCR validation, and functional validation.
