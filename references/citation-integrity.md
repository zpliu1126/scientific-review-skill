# Citation Integrity Rules

## Forbidden

- Creating plausible-looking citations from memory.
- Filling in missing DOI, PMID, authors, journal names, publication dates, species, cultivars, treatments, sample sizes, statistics, methods, pathways, or outcomes.
- Attributing a claim to a paper when the provided text does not support it.
- Treating a model's synthesis as if it were an author's stated conclusion.

## Required Labels

Use these labels when relevant:

- `Provided source`: a source supplied by the user or available in the working context.
- `Verified source`: a source checked through a reliable database or publication page.
- `User-provided note`: information supplied by the user, not independently verified.
- `Source-reported content`: methods, results, statistics, figure/table claims, and conclusions explicitly present in the source.
- `Reasonable inference`: cautious interpretation from one source, bounded by the provided evidence.
- `Model synthesis`: cross-source integration, conceptual framing, gap analysis, or proposed mechanism generated by the model.
- `[source needed]`: a claim that lacks source support and must not appear as a supported statement.
- `[needs verification]`: a claim linked to an available source or user note that still needs checking against the original source.

## Evidence Strength Heuristics

Prefer stronger evidence for stronger claims:

- Mechanistic claim: requires perturbation, rescue, dose-response, temporal evidence, or convergent assays.
- Association claim: can be supported by correlation, observational cohorts, or omics association, but should not be framed as causation.
- Candidate gene claim: can be supported by expression pattern, co-expression, annotation, and association, but requires functional validation for causal language.
- Translational claim: requires relevant human data, disease models, pharmacology, safety, or clinical context.
- Review-level consensus: requires multiple sources and acknowledgment of disagreement.

## Citation Format

Use the format requested by the user. If none is specified, use concise author-year citations when metadata is available:

```text
(Author et al., Year)
```

If metadata is incomplete:

```text
([source needed])
```

or

```text
(User-provided paper 1)
```
